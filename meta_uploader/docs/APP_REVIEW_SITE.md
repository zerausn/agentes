# App Review Site

Este subproyecto incluye un sitio estatico listo para publicarse gratis con
GitHub Pages.

## Rutas locales

- `meta_uploader/site/index.html`
- `meta_uploader/site/privacy.html`
- `meta_uploader/site/data-deletion.html`
- `meta_uploader/site/terms.html`

## URL esperada en GitHub Pages

Si el repo `zerausn/agentes` publica Pages como sitio de proyecto, las URLs
quedaran en este formato:

- `https://zerausn.github.io/agentes/`
- `https://zerausn.github.io/agentes/privacy.html`
- `https://zerausn.github.io/agentes/data-deletion.html`
- `https://zerausn.github.io/agentes/terms.html`

## Uso en Meta

Para Meta App Review conviene usar estas URLs limpias en lugar de enlaces tipo
`github.com/.../blob/...`.
